This project is designed as an educational tool to help users understand the fundamental mechanics of the stock market.
Learn how prices shift and practice tracking "Buy Prices" vs "Current Value" to see how net worth is calculated.
You get to experiment with different volatility settings (Low, Medium, High) to see how market swings affect your cash.
It Includes a 0.5% transaction fee to teach the reality of trading costs.
